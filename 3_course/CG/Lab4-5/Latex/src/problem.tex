\CWHeader{Лабораторная работа \textnumero 4-5}
\CWHeader{Ознакомление с технологией OpenGL.}
\CWProblem{
    Создать графическое приложение с использованием OpenGL. Используя результаты Л.Р.\textnumero3, изобразить
заданное тело (то же, что и в л.р. \textnumero3) с использованием средств OpenGL 2.1. Использовать буфер вершин. Точность
аппроксимации тела задается пользователем. Обеспечить возможность вращения и масштабирования многогранника
и удаление невидимых линий и поверхностей. Реализовать простую модель освещения на GLSL.
Параметры освещения и отражающие свойства материала задаются пользователем в диалоговом режиме.
}
